# 게임 코어 평가 및 발라트로 비교 분석

## 📊 현재 게임 코어 분석

### 현재 구현된 시스템

#### ✅ 구현 완료
1. **주사위 시스템**
   - 5개 주사위 고정 시작
   - 각 주사위는 6면체 (value 1-6, suit 4종)
   - 주사위 잠금/해제 기능
   - 리롤 시스템 (3회)

2. **족보 시스템**
   - 12종류의 족보 인식
   - 족보별 Multiplier 적용
   - Chips 계산 (주사위 값 합)

3. **전투 시스템**
   - 적 HP 시스템
   - 플레이어 HP 시스템
   - 스테이지 진행

4. **상점 시스템**
   - 조커 구매
   - 소모품 구매 (기본 구조만)

5. **주사위 덱 시스템**
   - 주사위 면 정보 저장 (diceDeck)
   - 인챈트를 통한 면 수정 가능

#### ⚠️ 부분 구현
1. **조커 시스템**
   - 데이터 구조는 있으나 실제 효과 적용 미구현
   - 데미지 계산에 조커 효과 미반영

2. **소모품 시스템**
   - 데이터 구조만 존재
   - 실제 사용 기능 미구현

3. **행성 카드 (족보 강화)**
   - handUpgrades 필드 존재
   - 실제 적용 로직 미구현

## 🔍 발라트로와의 차이점 분석

### 1. 주사위/카드 수량 시스템

#### 발라트로
- **카드 추가 가능**: 덱에 카드를 추가하여 핸드 크기 증가 가능
- **카드 제거 가능**: 소모품으로 카드 제거 가능
- **핸드 크기 변동**: 조커/바우처로 핸드 크기 변경 가능

#### 우리 게임 (현재)
- **주사위 5개 고정**: 게임 시작부터 끝까지 항상 5개
- **주사위 추가 불가**: 현재 시스템으로는 추가 불가능
- **주사위 제거 불가**: 파괴 메커니즘 없음

**⚠️ 걱정되는 점:**
- 발라트로의 핵심 재미 요소인 "덱 빌딩"이 부재
- 주사위 수량 조작이 불가능하여 전략 깊이가 얕아질 수 있음
- 조커 효과의 다양성이 제한됨 (예: "카드 추가" 효과 불가능)

---

### 2. 조커 발동 시스템

#### 발라트로
- **순차 발동**: 카드를 하나씩 플레이하며 각 카드마다 조커 효과 적용
- **조커 순서 중요**: 조커 위치에 따라 효과 순서 결정
- **재발동**: 특정 조커로 효과 재발동 가능

#### 우리 게임 (현재)
- **일괄 계산**: 모든 주사위를 한 번에 제출하고 데미지 계산
- **조커 순서 무관**: 조커 효과가 순차적으로 적용되지 않음
- **재발동 미구현**: 재발동 메커니즘 없음

**⚠️ 걱정되는 점:**
- 발라트로의 핵심 메커니즘인 "순차 발동"이 없어 전략 깊이 감소
- 조커 위치의 중요성이 사라짐
- 재발동 빌드가 불가능

---

### 3. 카드/주사위 타입 시스템

#### 발라트로
- **카드 타입**: Normal, Enhanced (Lucky, Mult, Bonus), Gold, Steel, Glass, Stone, Wild 등
- **Edition**: Foil, Holographic, Polychrome, Negative
- **Seal**: Gold Seal, Red Seal, Blue Seal, Purple Seal

#### 우리 게임 (현재)
- **주사위 타입 없음**: 모든 주사위가 동일
- **Edition 없음**: Foil 등 특수 효과 없음
- **Seal 없음**: 주사위에 특수 마크 불가능

**⚠️ 걱정되는 점:**
- 발라트로의 풍부한 카드 변형 시스템이 부재
- 전략 다양성 감소
- 수집/커스터마이징 재미 부족

---

### 4. 소모품 사용 시스템

#### 발라트로
- **즉시 사용**: 소모품을 즉시 사용하여 효과 발동
- **카드 변환**: 타로 카드로 카드 변환
- **족보 강화**: 행성 카드로 족보 강화
- **카드 생성**: 스펙트럴 카드로 카드 생성

#### 우리 게임 (현재)
- **소모품 사용 미구현**: 구매만 가능, 사용 불가
- **인챈트 시스템**: 주사위 면 수정 가능 (부분 구현)

**⚠️ 걱정되는 점:**
- 소모품이 게임플레이에 영향을 주지 않음
- 발라트로의 핵심 재미 요소인 "소모품 활용" 부재

---

### 5. 퀘스트/도전 과제 시스템

#### 발라트로
- **도전 과제**: 특정 조건 달성 시 보너스
- **시크릿 핸드**: 특수 족보 발견 시 보너스

#### 우리 게임 (현재)
- **퀘스트 시스템 없음**: 도전 과제나 특수 목표 없음

**⚠️ 걱정되는 점:**
- 장기 목표 부재로 반복 플레이 동기 감소
- 발견의 재미 부족
