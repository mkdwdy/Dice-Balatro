## 🎯 분석된 빌드 목록

### 1. 슈트 기반 빌드 (Suit-Based Builds)

#### Flush 빌드 (플러시 빌드)
- **등급**: A
- **난이도**: Medium
- **핵심 조커**: Greedy, Lusty, Wrathful, Gluttonous, Droll, Canio
- **전략**: 특정 슈트에 집중하여 Flush 족보를 자주 내기
- **예상 데미지**: 초반 50-100, 중반 200-500, 후반 1000+

### 2. 족보 기반 빌드 (Hand Type Builds)

#### Pair 빌드 (페어 빌드)
- **등급**: B
- **난이도**: Easy
- **핵심 조커**: Jolly Joker, Sly Joker
- **전략**: Pair 족보에 집중하는 초반 친화적 빌드
- **예상 데미지**: 초반 30-80, 중반 100-300, 후반 500-800

#### Straight 빌드 (스트레이트 빌드)
- **등급**: A
- **난이도**: Medium
- **핵심 조커**: Crazy Joker, Wily Joker, Shortcut, Four Fingers
- **전략**: Straight 족보에 집중, Shortcut과 Four Fingers로 쉽게 만들기
- **예상 데미지**: 초반 80-150, 중반 300-600, 후반 1000-2000

### 3. 성장형 빌드 (Scaling Builds)

#### Vampire + Midas 빌드
- **등급**: S
- **난이도**: Hard
- **핵심 조커**: Vampire, Midas Mask
- **전략**: 매 핸드마다 영구적으로 강해지는 성장형 빌드
- **예상 데미지**: 초반 50-100, 중반 300-800, 후반 5000+

#### Campfire Scaling 빌드
- **등급**: A
- **난이도**: Medium
- **핵심 조커**: Campfire, Hiker
- **전략**: 스테이지마다 강해지는 성장형 빌드
- **예상 데미지**: 초반 50-100, 중반 200-500, 후반 2000-5000

### 4. 복제 빌드 (Copy Builds)

#### Blueprint + Brainstorm 빌드
- **등급**: S+
- **난이도**: Expert
- **핵심 조커**: Blueprint, Brainstorm
- **전략**: 강력한 조커를 복제하여 시너지 극대화
- **예상 데미지**: 초반 100-200, 중반 500-1500, 후반 10000+

### 5. 재발동 빌드 (Retrigger Builds)

#### Triboulet Retrigger 빌드
- **등급**: S+
- **난이도**: Hard
- **핵심 조커**: Triboulet, Mime
- **전략**: 모든 효과를 두 번 발동시키는 재발동 빌드
- **예상 데미지**: 초반 100-300, 중반 1000-3000, 후반 10000+

### 6. 덱 압축 빌드 (Deck Compression Builds)

#### Baron King 빌드
- **등급**: S
- **난이도**: Hard
- **핵심 조커**: Baron, Mime
- **전략**: 특정 값(킹)에 집중하여 덱을 압축
- **예상 데미지**: 초반 100-200, 중반 500-2000, 후반 5000-20000

### 7. 경제 빌드 (Economy Builds)

#### Economy 빌드
- **등급**: B
- **난이도**: Easy
- **핵심 조커**: Golden Ticket, Business Card, Rough Gem
- **전략**: 골드를 많이 모아서 더 많은 조커 구매
- **예상 데미지**: 초반 30-60, 중반 100-300, 후반 500-1500

### 8. 특정 값 활용 빌드 (Value-Specific Builds)

#### Wee Joker 빌드
- **등급**: A
- **난이도**: Medium
- **핵심 조커**: Wee Joker, Hack
- **전략**: 낮은 값(2)을 활용하는 빌드
- **예상 데미지**: 초반 50-100, 중반 300-800, 후반 2000-5000

### 9. 하이브리드 빌드 (Hybrid Builds)

#### Hybrid Flush + Straight 빌드
- **등급**: S
- **난이도**: Expert
- **핵심 조커**: Droll, Crazy, Canio, Shortcut, Four Fingers
- **전략**: Straight Flush를 목표로 하는 하이브리드 빌드
- **예상 데미지**: 초반 100-200, 중반 500-1500, 후반 5000-20000

- 재발동: 1개
- 덱 압축: 1개
- 경제: 1개
- 특정 값 활용: 1개
- 하이브리드: 1개

### 등급별 분포

- S: 3개 (Vampire+Midas, Baron King, Hybrid Flush+Straight)
- A: 3개 (Flush, Straight, Campfire Scaling, Wee Joker)
- B: 2개 (Pair, Economy)

### 난이도별 분포
- Easy: 2개 (Pair, Economy)
- Medium: 4개 (Flush, Straight, Campfire Scaling, Wee Joker)
- Hard: 3개 (Vampire+Midas, Triboulet Retrigger, Baron King)
- Expert: 2개 (Blueprint+Brainstorm, Hybrid Flush+Straight)

## 🔄 게임플레이 통합 워크플로우

### Phase 1: 빌드 발견
- 상점에서 현재 보유 조커 기반 빌드 추천
- 조커 구매 시 빌드 정보 표시
- 게임 화면에 빌드 진행도 표시

### Phase 2: 빌드 구성
- 조커 슬롯에 빌드별 색상 코딩
- 조커 간 시너지 표시
- 빌드 완성 알림

### Phase 3: 빌드 실행
- 자동 효과 계산 (Chips → Mult → Retrigger)
- 빌드별 최적 족보 제안
- 빌드 성장 표시

### Phase 4: 빌드 전환
- 빌드 전환 제안
- 조커 판매 가이드

## 🎮 주사위 게임 적용 방법

### 주요 변환 사항
1. **슈트 기반 빌드**: 주사위 인챈트로 같은 슈트 설정
2. **족보 기반 빌드**: 주사위 값 조합으로 족보 만들기
3. **성장형 빌드**: 매 핸드/스테이지마다 영구 증가
4. **복제 빌드**: 조커 위치 관리 중요
5. **재발동 빌드**: 효과를 두 번 적용
6. **덱 압축 빌드**: 주사위 면을 특정 값으로 고정
7. **경제 빌드**: 골드 생성 → 조커 구매
8. **특정 값 활용**: 값별 조커와 조합

## 💡 핵심 인사이트

### 발라트로의 재미 요소
1. **시너지 발견**: 여러 조커가 조합되어 강해지는 순간
2. **전략적 선택**: 제한된 슬롯과 골드로 최적 조합 선택
3. **성장의 재미**: 시간이 지날수록 강해지는 만족감
4. **빌드 완성**: 목표 빌드를 완성했을 때의 성취감

### 우리 게임에 적용할 점
1. **빌드 발견의 즐거움**: 상점에서 빌드 힌트 제공
2. **시각적 피드백**: 빌드 진행도와 시너지 표시
3. **전략적 깊이**: 다양한 빌드로 다양한 플레이 스타일 지원
4. **성장의 만족감**: 성장형 빌드로 후반 강력함 체험
