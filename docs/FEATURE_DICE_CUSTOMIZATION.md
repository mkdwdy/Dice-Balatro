# 피쳐: 주사위 커스터마이징 시스템 (소모품 기반)

## 📋 개요

주사위 추가 시에는 랜덤화되어 등장하지만, 게임플레이 도중 소모품을 사용하여 주사위의 속성을 전략적으로 변경할 수 있는 시스템입니다.

---

## 🎯 핵심 원칙

1. **주사위 추가 = 랜덤화**: 모든 주사위는 랜덤 속성으로 등장
2. **커스터마이징 = 소모품**: 게임플레이 중 소모품으로만 변경 가능
3. **전략적 선택**: 소모품 사용은 제한적이므로 신중한 선택 필요

---

## 🔧 커스터마이징 타입

### 1. 타입 변경 (Material Type Change)

#### 소모품 예시
- **"Material Transmutation" 타로 카드**: 주사위 타입 변경
  - Normal → Glass/Stone/Steel 중 랜덤
  - Glass → Stone/Steel/Gold 중 랜덤
  - 등등

#### 사용 방법
1. 소모품 선택
2. 주사위 선택
3. 새로운 타입 선택 (또는 랜덤)
4. 적용

#### 효과
- 주사위의 특수 능력 변경
- 예: Normal → Glass (Mult ×2, 파괴 확률 20%)

---

### 2. Foil 추가/변경 (Foil Enchant)

#### 소모품 예시
- **"Foil Enchant" 타로 카드**: Foil 추가
  - 없음 → Foil/Holographic/Polychrome 중 선택 또는 랜덤
- **"Foil Upgrade" 스펙트럴 카드**: Foil 업그레이드
  - Foil → Holographic
  - Holographic → Polychrome

#### 사용 방법
1. 소모품 선택
2. 주사위 선택
3. Foil 타입 선택 (또는 랜덤)
4. 적용

#### 효과
- Mult/Chips 보너스 추가
- 예: 없음 → Polychrome (Mult ×1.5)

---

### 3. 슈트 변경 (Suit Change)

#### 소모품 예시
- **"Suit Change" 타로 카드**: 슈트 변경
  - 전체 면 슈트 변경 또는 특정 면만 변경
- **"Suit Unification" 스펙트럴 카드**: 모든 면을 같은 슈트로

#### 사용 방법
1. 소모품 선택
2. 주사위 선택
3. 변경할 면 선택 (전체 또는 특정 면)
4. 새 슈트 선택
5. 적용

#### 효과
- 족보 계산에 영향
- 예: Flush 빌드를 위해 모든 면을 ♥로 변경

---

### 4. 면 값 변경 (Face Value Change)

#### 소모품 예시
- **"Face Enchant" 타로 카드**: 면 값 변경
  - 특정 면의 값 변경
- **"Value Boost" 타로 카드**: 면 값 증가
  - 값 +1 (최대 6)
- **"Perfect Dice" 스펙트럴 카드**: 모든 면을 같은 값으로

#### 사용 방법
1. 소모품 선택
2. 주사위 선택
3. 변경할 면 선택 (1개 또는 여러 개)
4. 새 값 선택 (또는 +1/-1)
5. 적용

#### 효과
- 족보 계산에 영향
- 예: 모든 면을 6으로 변경하여 Yahtzee 빌드

---

## 🎮 게임플레이 예시

### 시나리오 1: Flush 빌드 완성
1. 랜덤 주사위 3개 획득 (슈트가 각각 다름)
2. "Suit Unification" 스펙트럴 카드 사용
3. 주사위 3개 모두 ♥ 슈트로 변경
4. Flush 빌드 완성

### 시나리오 2: Glass 주사위 강화
1. 랜덤으로 Glass 주사위 획득 (Foil 없음)
2. "Foil Enchant" 타로 카드 사용
3. Polychrome Foil 추가
4. Mult ×2 (Glass) × 1.5 (Polychrome) = Mult ×3

### 시나리오 3: Yahtzee 빌드
1. 랜덤 주사위 획득 (값이 다양함)
2. "Perfect Dice" 스펙트럴 카드 사용
3. 모든 면을 6으로 변경
4. Yahtzee 빌드 완성
