# 피쳐: 상점 주사위 판매 및 소모성 주사위 시스템

## 📋 개요
상점에서 주사위를 구매할 수 있으며, 저렴한 주사위는 소모성(내구도/파괴 확률)을 가지고, 인챈트로 영구화 가능한 시스템.

## 🎯 목표
- 주사위를 상점에서 구매 가능
- 소모성 주사위로 리스크/리워드 제공
- 인챈트로 영구화하여 진행감 제공

## 🔧 설계

### 주사위 타입
```typescript
type DiceType = 
  | 'permanent'    // 영구 주사위 (비쌈)
  | 'durability'   // 내구도 주사위 (X회 사용)
  | 'fragile';     // 파괴 확률 주사위 (매번 확률)

interface ShopDice {
  id: string;
  name: string;
  type: DiceType;
  cost: number;
  durability?: number;      // 내구도 (durability 타입)
  destroyChance?: number;   // 파괴 확률 (fragile 타입, 0-1)
  baseStats: {
    faces: DiceFace[];      // 기본 면 구성
  };
}
```

### 인챈트 업그레이드
- 소모성 주사위 → 영구 주사위 변환
- 비용: 주사위 구매 가격의 50-100%

## ⚖️ 밸런싱
- 영구 주사위: $15-25
- 내구도 주사위: $5-10 (내구도 3-5회)
- 파괴 확률 주사위: $3-7 (파괴 확률 10-30%)

## 🚀 구현 단계
1. 상점에 주사위 판매 추가
2. 주사위 타입 시스템 구현
3. 내구도/파괴 확률 로직 구현
4. 인챈트 업그레이드 시스템 구현
