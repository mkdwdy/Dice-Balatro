# Cursor 토큰 사용량 표시 위치 및 조건

## ⚠️ 중요: 제가 직접 확인할 수 없습니다

제가 접근할 수 있는 정보에는 Cursor의 토큰 사용량 표시 위치가 포함되어 있지 않습니다.
아래는 일반적인 IDE 패턴과 Cursor의 가능한 표시 위치에 대한 정보입니다.

## 🔍 가능한 표시 위치

### 1. 상태 바 (Status Bar) - 화면 하단
**위치**: Cursor 창 맨 아래쪽 상태 바

**표시되는 경우**:
- 토큰 사용량이 설정에서 활성화된 경우
- 실시간 사용량을 표시하도록 설정된 경우
- 일일/월간 제한에 근접한 경우

**표시되지 않는 경우**:
- 설정에서 비활성화된 경우
- 사용량이 매우 낮아 표시할 필요가 없는 경우
- UI 설정에 따라 숨겨진 경우

**확인 방법**:
1. Cursor 창 하단을 확인
2. 상태 바에 숫자나 진행 바가 있는지 확인
3. 상태 바 항목을 클릭하여 상세 정보 확인

### 2. 설정 메뉴
**위치**: 
- `Cmd/Ctrl + ,` → Settings
- 또는 메뉴: Cursor → Settings

**확인 가능한 정보**:
- 현재 토큰 사용량
- 일일/월간 제한
- 사용량 통계
- 리셋 시간

### 3. 계정/프로필 메뉴
**위치**:
- Cursor 메뉴 → Account
- 또는 우측 상단 프로필 아이콘

**확인 가능한 정보**:
- 구독 정보
- 토큰 사용량
- 사용량 히스토리

### 4. 명령 팔레트
**위치**: `Cmd/Ctrl + Shift + P`

**명령어**:
- "Show Usage" 또는 "Usage"
- "Account" 또는 "Subscription"

## 📊 표시 조건

### 항상 표시되는 경우
- ✅ 설정에서 "Show token usage" 활성화
- ✅ 토큰 제한에 근접 (예: 80% 이상 사용)
- ✅ 프리미엄 기능 사용 중

### 조건부 표시되는 경우
- ⚠️ 사용량이 특정 임계값을 넘은 경우
- ⚠️ 일일 제한에 근접한 경우
- ⚠️ 월간 제한에 근접한 경우

### 표시되지 않는 경우
- ❌ 설정에서 비활성화
- ❌ 사용량이 매우 낮음
- ❌ 무료 플랜에서 일부 정보 제한

## 🔧 확인 방법 (단계별)

### 방법 1: 상태 바 확인
```
1. Cursor 창 하단 상태 바 확인
2. 숫자나 진행 바가 있는지 확인
3. 클릭하여 상세 정보 확인
```

### 방법 2: 설정에서 확인
```
1. Cmd/Ctrl + , (설정 열기)
2. "Usage" 또는 "Account" 검색
3. 토큰 사용량 섹션 확인
```

### 방법 3: 명령 팔레트
```
1. Cmd/Ctrl + Shift + P
2. "usage" 또는 "account" 입력
3. 관련 명령어 선택
```

### 방법 4: 웹 대시보드
```
1. https://cursor.sh 접속
2. 로그인
3. Account 또는 Usage 섹션 확인
```

## 💡 실제 확인 방법

가장 확실한 방법은:

1. **Cursor 설정 직접 확인**
   - `Cmd/Ctrl + ,` → Settings
   - "usage", "token", "account" 검색

2. **Cursor 메뉴 확인**
   - 상단 메뉴바 → Cursor → Account/Usage

3. **상태 바 우클릭**
   - 하단 상태 바를 우클릭하여 표시 옵션 확인
