# 순차 발동 시스템 테스트 가이드

## 🚀 서버 실행

개발 서버가 실행 중입니다:
- **URL**: http://localhost:5000
- **API**: http://localhost:5000/api/*

## 🧪 테스트 시나리오

### 1. 기본 순차 발동 테스트

1. 게임 시작
   - http://localhost:5000 접속
   - "New Game" 버튼 클릭

2. 주사위 굴리기
   - "Roll" 버튼 클릭하여 주사위 굴리기
   - 원하는 주사위를 클릭하여 잠금 (최소 2개 이상)

3. 핸드 제출
   - 잠긴 주사위가 있으면 자동으로 족보 인식
   - "Submit" 버튼 클릭
   - 순차 발동이 실행되어야 함

### 2. 조커 효과 테스트

1. 상점에서 조커 구매
   - 게임 진행 중 상점 진입
   - "Joker" 조커 구매 (+4 Mult)
   - "Greedy Joker" 구매 (♦ 슈트마다 +3 Mult)

2. 조커 효과 확인
   - 주사위 굴리기
   - ♦ 슈트 주사위를 잠금
   - 핸드 제출
   - 순차 발동 시 각 주사위마다 조커 효과가 적용되어야 함

### 3. 순차 발동 순서 확인

1. 여러 주사위 잠금
   - 주사위 5개 모두 잠금
   - 핸드 제출

2. 발동 순서 확인
   - 주사위가 왼쪽부터 순차적으로 발동되어야 함
   - 각 주사위마다 조커 효과가 적용되어야 함

## 📊 예상 동작

### 순차 발동 로직
- 잠긴 주사위를 왼쪽부터 순차적으로 발동
- 각 주사위 발동 시:
  - 기본 Chips = 주사위 값
  - 조커 효과 적용 (순서대로)
  - Chips와 Multiplier 누적

### 데미지 계산
- 최종 데미지 = 전체 Chips × (전체 Multiplier + 1)
- 족보 Multiplier는 기본 Multiplier에 포함

## ⚠️ 알려진 이슈

1. **Multiplier 계산 문제**
   - 현재 각 주사위마다 Multiplier를 누적하고 있음
   - 발라트로 방식: 각 주사위의 Chips를 누적하고, Multiplier는 전체에 한 번만 적용
   - 수정 필요

2. **애니메이션 미구현**
   - 순차 발동 애니메이션이 아직 구현되지 않음
   - 현재는 즉시 데미지 계산만 수행

3. **조커 효과 제한적**
   - 현재 기본 조커 효과만 구현됨
   - Phase 1-2에서 확장 예정
